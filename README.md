# 📚 Book Scraper

## Description

This is a Python project that scrapes book titles, prices, ratings, and images from an online bookstore.

## Features

* Extracts book title, price, and rating
* Downloads book images
* Handles multiple pages (pagination)
* Saves data into CSV file

## Tools Used

* Python
* Selenium
* Requests

## How to Run

1. Open Command Prompt (or terminal) in the `book_scraper.py` folder
2. Install required libraries if not already installed:

   ```bash
   pip install selenium requests
   ```
3. Run the script:

   ```bash
   python basi_selen.py
   ```

## Output

* `books.csv` → contains scraped data
* `images/` → contains downloaded images
